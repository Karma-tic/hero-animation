# Scroll-Driven Hero Section Animation

This project demonstrates a scroll-driven hero animation built using:

- Next.js
- Tailwind CSS
- GSAP
- ScrollTrigger

## Features

- Pinned scroll section
- Car moves dynamically based on scroll progress
- Progressive letter reveal
- Expanding motion trail
- Floating metric boxes with scroll-based animations
- Fully responsive layout

## Tech Stack

- Next.js
- Tailwind CSS
- GSAP (GreenSock Animation Platform)
